This was a site I built for a friend who wanted to store a database of historic trombones made before 1800. I used React.js, Express.js, Materialize, MongoDB, Heroku, and AWS S3.

This was my third project using Express.js, MongoDB, React.js, Materialize, and Auth0 for authentication. This app is a simple landlord/tenant app that keeps track of property details, assigned tenants, and facilitates service requests.
